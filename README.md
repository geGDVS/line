# 直线方程可视化与转换工具

一个基于 HTML、CSS、JavaScript 开发的直线方程可视化工具，支持多种直线方程形式的相互转换和图形化展示。

## 功能特点

### 🔧 支持的方程形式
- 斜截式 (y = mx + b)
- 点斜式 (y - y₀ = m(x - x₀))
- 两点式 ((y-y₁)/(y₂-y₁) = (x-x₁)/(x₂-x₁))
- 截距式 (x/a + y/b = 1)
- 点方向式 ((x-x₀)/u = (y-y₀)/v)
- 点法向式 (a(x-x₀) + b(y-y₀) = 0)
- 一般式 (Ax + By + C = 0)

### 📊 可视化展示
- 实时绘制直线图形
- 显示关键点（截距点、参考点）
- 展示方向向量和法向量
- 鼠标悬浮显示坐标信息

### 🎨 交互特性
- 选项卡式参数输入
- 快速示例一键加载
- 响应式布局，适配移动端
- 平滑的过渡动画效果

## 使用方法

1. 将 `index.html`、`styles.css`、`script.js` 放在同一目录下
2. 用浏览器打开 `index.html`
3. 选择任意方程形式，输入参数后点击「计算并显示直线」
4. 查看可视化图形和所有形式的转换结果
